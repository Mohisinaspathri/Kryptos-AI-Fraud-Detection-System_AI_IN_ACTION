# Kryptos-AI-Fraud-Detection-System_AI_IN_ACTION
A machine learning-based fraud detection system using Random Forest that provides real-time predictions, risk scoring, and explainable results

This project is an AI-based fraud detection system designed to identify suspicious financial transactions in real-time. It leverages machine learning to analyze transaction patterns, user behavior, and risk indicators to predict the likelihood of fraud.




Financial fraud is a major challenge in digital transactions. Due to highly imbalanced data (very few fraud cases), traditional methods often fail to detect fraud effectively.



We built a machine learning model using *Random Forest* that:
- Predicts fraud probability
- Assigns a risk score
- Provides explainable insights
- Enables real-time detection through a web interface




- 🔢 *Fraud Risk Score (0–100)*
- 🟢🟡🔴 *Risk Classification (Low / Medium / High)*
- 🧠 *Explainable AI (Reasons for prediction)*
- 💳 *User-friendly Input Interface*
- ⚡ *Real-time Fraud Detection (Streamlit App)*
- 🏪 *Merchant Category Selection (MCC Mapping)*
